# RP2040
This project is an attempt to make a hal(hardware abstraction layer) and a RTOS for the Raspberry Pi Pico (RP2040 arm-cortex-m0+) microcontroller.

## Inspirations
A lot of the code and api will be inspired by microzig and I'll try to make sure that all the licenses are properly followed.

## AI Usage
There will be some AI usage. I am not very familiar with writing embedded code and I sometimes use AI to learn things and generate example snippets. For e.g. the ./bootloader/linker.ld was written by gemini 3.1 pro as I was not familiar with linker scripts.
